# dev-workflow

## Commands

### review-pr

**Full review (default):**

```
/review-pr
```

**Specific aspects:**

```
/review-pr tests errors
# Reviews only test coverage and error handling

/review-pr comments
# Reviews only code comments

/review-pr simplify
# Simplifies code after passing review
```

**Parallel review:**

```
/review-pr all parallel
# Launches all agents in parallel
```
